### 실행 방법
**Prerequisites**
- Node.js >= 16
- Git
- 백엔드 서버 실행 (⚠️ 반드시 함께 구동해야 전체 기능 사용 가능)
  </br>
  백엔드 레포: https://github.com/4LINETHON-Team1/Cooltime-BE

**1. 프로젝트 클론**
```bash
git clone https://github.com/4LINETHON-Team1/Cooltime-FE
cd cooltime
```

**2. 필요한 라이브러리 설치**
```bash
npm install
```

**3. 환경변수 설정**
프로젝트 루트에 `.env` 파일을 생성하고 다음 정보를 입력:
```env
# 백엔드 서버 설정
VITE_API_BASE_URL=your-backend-server-url
```

**4. 개발 서버 실행**
```bash
npm run dev
```
서버가 실행되면 http://localhost:5173 (기본 포트)에서 접속할 수 있습니다.
